IMAGES FOLDER
=============

Place the following images in this folder:

1. logo.png
   - The SimplifyMVT Academy logo file
   - The logo image from the PDF / brand assets should be placed here
   - Recommended size: Minimum 200px height, PNG with transparent background
   - The HTML has fallback text if logo.png is not found

2. (Optional) hero-bg.jpg
   - A background image for the hero section
   - Currently using CSS gradient - can be replaced with a real image

HOW TO USE:
- Copy your logo file and rename it to "logo.png"
- Place it in this directory: src/assets/images/logo.png
- The website will automatically display it in the navbar, hero card, and footer

NOTE: The website is fully functional even without the logo image file.
A CSS-based fallback logo will be shown automatically.
